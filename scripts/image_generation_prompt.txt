【Visual System - Minimalist Art Aesthetic】

Color Palette:
- Primary: Deep tech blue (#2B5F8E) + electric accent (#00D4FF)
- Secondary: Earth tones (off-white #FAFAFA, warm gray #E8E8E8, graphite #3A3A3A)
- Accent: Single restrained highlight color
- Background: Pure white or very light gray (#FAFAFA)
- Text: Dark gray not pure black (#2C2C2C)

Typography System:
- Headings: Ultra-light sans-serif (Helvetica Neue UltraLight / SF Pro Display Ultralight)
- Body: Modern geometric sans (Inter / SF Pro Text)
- Data: Monospace (SF Mono / Roboto Mono)
- Font size hierarchy: 6-level system (48/36/28/18/14/11pt)

Icon System:
- Consistent 2pt stroke width
- Rounded corners consistent (2px)
- 24x24px grid alignment
- Stroke-based not fill-based
- Negative space utilization

Layout System:
- 12-column grid system
- 8pt baseline grid alignment
- Module spacing: 24pt/32pt/48pt
- Margins: 48pt minimum page margins
- Card border-radius: 8-12px
- Subtle dividers: 0.5pt, 15% opacity

Texture Details:
- Micro-gradient backgrounds (3-5% color variation)
- Soft shadows (0 4px 24px rgba(0,0,0,0.04))
- Frosted glass effects (backdrop-filter: blur(20px))
- Subtle elevation differences between elements
- Data areas with light fill (5% opacity)

【Information Visualization Principles】
- Data charts: Edward Tufte minimal data-ink ratio
- Remove unnecessary decorations
- Chart grid lines very subtle (10% opacity)
- Data points precisely labeled
- Smooth natural trend lines

【Breathing Room Creation】
- White space between modules >= 25% of module content width
- Text line-height 1.6-1.8x
- Paragraph spacing >= 2 line-heights
- Image-text combination white space rules
